Material 组件概述

Flutter 提供了各种视觉、行为和动效丰富的小部件，它们实现了 Material 3 设计规范。Material 3 是 Flutter 的默认设计语言，能帮助你设计和构建美观、实用的应用程序，并可适应任何平台。

Flutter 3.16 已完成将 Material 3 作为默认设计的转换。本文档将详细介绍各种 Material 组件的使用方法、配置选项和最佳实践。

快速开始

基础设置

import 'package:flutter/material.dart';

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({Key? key}) : super(key: key);

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Flutter Demo',
      theme: ThemeData(
        colorScheme: ColorScheme.fromSeed(seedColor: Colors.deepPurple),
        useMaterial3: true,
      ),
      home: const MyHomePage(title: 'Flutter Demo Home Page'),
    );
  }
}

class MyHomePage extends StatefulWidget {
  const MyHomePage({Key? key, required this.title}) : super(key: key);

  final String title;

  @override
  State<MyHomePage> createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        backgroundColor: Theme.of(context).colorScheme.inversePrimary,
        title: Text(widget.title),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: const <Widget>[
            Text(
              'Hello, Flutter!',
            ),
          ],
        ),
      ),
    );
  }
}

AppBar

AppBar 是一个通常显示在屏幕顶部的应用栏，用于显示应用的标题、操作按钮和导航选项。

基本用法

AppBar(
  title: Text('应用标题'),
  actions: [
    IconButton(
      icon: Icon(Icons.search),
      onPressed: () {},
    ),
    IconButton(
      icon: Icon(Icons.more_vert),
      onPressed: () {},
    ),
  ],
)

常用属性

属性 类型 说明
title Widget 应用栏的标题
actions List<Widget> 右侧的操作按钮
leading Widget 左侧的控件，通常是返回按钮
backgroundColor Color 应用栏的背景色
elevation double 应用栏的阴影高度

示例

Scaffold(
  appBar: AppBar(
    title: Text('复杂的 AppBar'),
    leading: IconButton(
      icon: Icon(Icons.menu),
      onPressed: () {},
    ),
    actions: [
      IconButton(
        icon: Icon(Icons.search),
        onPressed: () {},
      ),
      IconButton(
        icon: Icon(Icons.favorite),
        onPressed: () {},
      ),
      PopupMenuButton(
        itemBuilder: (context) => [
          PopupMenuItem(child: Text('设置')),
          PopupMenuItem(child: Text('关于')),
        ],
      ),
    ],
    backgroundColor: Colors.blue,
    elevation: 4.0,
  ),
  body: Center(
    child: Text('AppBar 示例'),
  ),
)

Card

Card 是一个带有阴影和圆角的容器，用于显示相关信息的分组。

基本用法

Card(
  child: Column(
    children: [
      ListTile(
        leading: Icon(Icons.album),
        title: Text('卡片标题'),
        subtitle: Text('卡片副标题'),
      ),
      Padding(
        padding: EdgeInsets.all(16.0),
        child: Text('卡片内容区域'),
      ),
      ButtonBar(
        children: [
          TextButton(
            onPressed: () {},
            child: Text('操作 1'),
          ),
          TextButton(
            onPressed: () {},
            child: Text('操作 2'),
          ),
        ],
      ),
    ],
  ),
)

常用属性

属性 类型 说明
child Widget 卡片的内容
elevation double 卡片的阴影高度
shape ShapeBorder 卡片的形状
margin EdgeInsetsGeometry 卡片的外边距
color Color 卡片的背景色

Dialog

Dialog 是一个模态窗口，用于向用户显示重要信息或请求用户做出决策。

基本用法

Future _showMyDialog() async {
  return showDialog(
    context: context,
    barrierDismissible: false, // 用户必须点击按钮才能关闭
    builder: (BuildContext context) {
      return AlertDialog(
        title: const Text('对话框标题'),
        content: const SingleChildScrollView(
          child: ListBody(
            children: [
              Text('这是对话框内容'),
              Text('可以显示多行文本或其他组件'),
            ],
          ),
        ),
        actions: [
          TextButton(
            child: const Text('取消'),
            onPressed: () {
              Navigator.of(context).pop();
            },
          ),
          TextButton(
            child: const Text('确定'),
            onPressed: () {
              Navigator.of(context).pop();
            },
          ),
        ],
      );
    },
  );
}

显示对话框

ElevatedButton(
  onPressed: _showMyDialog,
  child: const Text('显示对话框'),
)

FloatingActionButton

FloatingActionButton (FAB) 是一个悬浮在界面上的圆形按钮，通常用于触发主要操作。

基本用法

Scaffold(
  appBar: AppBar(
    title: Text('FAB 示例'),
  ),
  body: Center(
    child: Text('点击右下角的按钮'),
  ),
  floatingActionButton: FloatingActionButton(
    onPressed: () {
      // 处理点击事件
    },
    tooltip: '添加',
    child: Icon(Icons.add),
  ),
  floatingActionButtonLocation: FloatingActionButtonLocation.centerFloat,
)

常用属性

属性 类型 说明
onPressed VoidCallback 点击事件处理函数
child Widget 按钮内的子组件，通常是图标
tooltip String 长按显示的提示文本
backgroundColor Color 按钮的背景色
shape ShapeBorder 按钮的形状

ListView

ListView 是一个可以垂直滚动的列表组件，用于显示大量数据。

基本用法

// 基本列表
ListView(
  children: const [
    ListTile(
      leading: Icon(Icons.map),
      title: Text('地图'),
    ),
    ListTile(
      leading: Icon(Icons.phone),
      title: Text('电话'),
    ),
    ListTile(
      leading: Icon(Icons.photo_album),
      title: Text('相册'),
    ),
  ],
)

// 构建器模式（适用于大量数据）
ListView.builder(
  itemCount: items.length,
  itemBuilder: (context, index) {
    return ListTile(
      title: Text('项目 ${items[index]}'),
    );
  },
)

// 分隔列表
ListView.separated(
  itemCount: 100,
  separatorBuilder: (context, index) => Divider(),
  itemBuilder: (context, index) {
    return ListTile(
      title: Text('项目 $index'),
    );
  },
)

常用属性

属性 类型 说明
scrollDirection Axis 滚动方向
padding EdgeInsetsGeometry 内边距
reverse bool 是否反向滚动
shrinkWrap bool 是否根据内容确定高度
physics ScrollPhysics 滚动行为

TextField

TextField 是一个文本输入框，用于获取用户的文本输入。

基本用法

TextField(
  decoration: InputDecoration(
    border: OutlineInputBorder(),
    labelText: '请输入文本',
    hintText: '提示文本',
    prefixIcon: Icon(Icons.search),
  ),
  onChanged: (text) {
    // 文本变化时的处理
  },
  onSubmitted: (text) {
    // 提交时的处理
  },
  obscureText: false, // 是否是密码输入
  keyboardType: TextInputType.text, // 键盘类型
)

常用属性

属性 类型 说明
controller TextEditingController 文本控制器
decoration InputDecoration 输入框装饰
keyboardType TextInputType 键盘类型
obscureText bool 是否隐藏文本
maxLength int 最大字符长度
onChanged ValueChanged<String> 文本变化回调
validator FormFieldValidator<String> 表单验证器

带控制器的TextField

class _MyHomePageState extends State {
  final TextEditingController _controller = TextEditingController();

  @override
  void dispose() {
    _controller.dispose();
    super.dispose();
  }

  @override
  Widget build(BuildContext context) {
    return TextField(
      controller: _controller,
      decoration: InputDecoration(
        labelText: '请输入您的名字',
      ),
    );
  }
}

BottomNavigationBar

BottomNavigationBar 是一个显示在屏幕底部的导航栏，用于在不同的主要视图之间切换。

基本用法

class _MyHomePageState extends State {
  int _selectedIndex = 0;

  static const List _widgetOptions = [
    Text('首页'),
    Text('搜索'),
    Text('设置'),
  ];

  void _onItemTapped(int index) {
    setState(() {
      _selectedIndex = index;
    });
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: const Text('底部导航栏示例'),
      ),
      body: Center(
        child: _widgetOptions.elementAt(_selectedIndex),
      ),
      bottomNavigationBar: BottomNavigationBar(
        items: const [
          BottomNavigationBarItem(
            icon: Icon(Icons.home),
            label: '首页',
          ),
          BottomNavigationBarItem(
            icon: Icon(Icons.search),
            label: '搜索',
          ),
          BottomNavigationBarItem(
            icon: Icon(Icons.settings),
            label: '设置',
          ),
        ],
        currentIndex: _selectedIndex,
        selectedItemColor: Colors.blue,
        onTap: _onItemTapped,
      ),
    );
  }
}

常用属性

属性 类型 说明
items List<BottomNavigationBarItem> 导航项列表
currentIndex int 当前选中项的索引
onTap ValueChanged<int> 点击事件回调
selectedItemColor Color 选中项的颜色
unselectedItemColor Color 未选中项的颜色
type BottomNavigationBarType 导航栏类型

Button

Flutter 提供了多种按钮组件，用于响应用户的点击操作。

基本用法

// ElevatedButton - 带有背景色和阴影的按钮
ElevatedButton(
  onPressed: () {
    // 处理点击事件
  },
  child: Text('Elevated Button'),
  style: ElevatedButton.styleFrom(
    backgroundColor: Colors.blue,
    padding: EdgeInsets.symmetric(horizontal: 24, vertical: 12),
    textStyle: TextStyle(fontSize: 16),
  ),
)

// TextButton - 文本按钮，没有背景色
TextButton(
  onPressed: () {
    // 处理点击事件
  },
  child: Text('Text Button'),
  style: TextButton.styleFrom(
    foregroundColor: Colors.blue,
  ),
)

// OutlinedButton - 带边框的按钮
OutlinedButton(
  onPressed: () {
    // 处理点击事件
  },
  child: Text('Outlined Button'),
  style: OutlinedButton.styleFrom(
    foregroundColor: Colors.blue,
    side: BorderSide(color: Colors.blue),
  ),
)

// IconButton - 图标按钮
IconButton(
  icon: Icon(Icons.favorite),
  onPressed: () {
    // 处理点击事件
  },
  tooltip: '收藏',
)

// 禁用状态
ElevatedButton(
  onPressed: null, // 设置为null表示禁用
  child: Text('禁用按钮'),
)

按钮样式

样式属性 说明
backgroundColor 按钮的背景色
foregroundColor 按钮的前景色（文本和图标）
padding 按钮的内边距
textStyle 文本样式
shape 按钮的形状
elevation 阴影高度（仅 ElevatedButton）

自定义按钮形状

ElevatedButton(
  onPressed: () {},
  child: Text('圆角按钮'),
  style: ElevatedButton.styleFrom(
    shape: RoundedRectangleBorder(
      borderRadius: BorderRadius.circular(20),
    ),
  ),
)

// 圆形按钮
ElevatedButton(
  onPressed: () {},
  child: Icon(Icons.add),
  style: ElevatedButton.styleFrom(
    shape: CircleBorder(),
    padding: EdgeInsets.all(20),
  ),
)
