Material 组件概述

Flutter 提供了各种视觉、行为和动效丰富的小部件，它们实现了 Material 3 设计规范。Material 3 是 Flutter 的默认设计语言，能帮助你设计和构建美观、实用的应用程序，并可适应任何平台。

Flutter 3.16 已完成将 Material 3 作为默认设计的转换。本文档将详细介绍各种 Material 组件的使用方法、配置选项和最佳实践。

快速开始

基础设置

import 'package:flutter/material.dart';

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({Key? key}) : super(key: key);

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Flutter Demo',
      theme: ThemeData(
        colorScheme: ColorScheme.fromSeed(seedColor: Colors.deepPurple),
        useMaterial3: true,
      ),
      home: const MyHomePage(title: 'Flutter Demo Home Page'),
    );
  }
}

class MyHomePage extends StatefulWidget {
  const MyHomePage({Key? key, required this.title}) : super(key: key);

  final String title;

  @override
  State<MyHomePage> createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        backgroundColor: Theme.of(context).colorScheme.inversePrimary,
        title: Text(widget.title),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: const <Widget>[
            Text(
              'Hello, Flutter!',
            ),
          ],
        ),
      ),
    );
  }
}

        icon: Icon(Icons.search),
        label: '搜索',

        icon: Icon(Icons.favorite),
        label: '收藏',
      ),
      NavigationDestination(
        icon: Icon(Icons.person),

      ),
    ],

FloatingActionButton

FloatingActionButton 是一个可点击的块，包含一个图标，使关键操作始终触手可及。它通常用于触发主要操作。

基本用法

常用属性

属性 类型 说明

child Widget 按钮内的图标或其他组件
backgroundColor Color 按钮的背景色
foregroundColor Color 按钮内容的颜色
shape ShapeBorder 按钮的形状

示例

Scaffold(

)

ListView

ListView 是一个可以垂直滚动的列表组件，用于显示大量数据。

基本用法

// 基本列表
ListView(
  children: const [
    ListTile(
      leading: Icon(Icons.map),
      title: Text('地图'),
    ),
    ListTile(
      leading: Icon(Icons.phone),
      title: Text('电话'),
    ),
    ListTile(
      leading: Icon(Icons.photo_album),
      title: Text('相册'),
    ),
  ],
)

// 构建器模式（适用于大量数据）
ListView.builder(
  itemCount: items.length,
  itemBuilder: (context, index) {
    return ListTile(
      title: Text('项目 ${items[index]}'),
    );
  },
)

// 分隔列表
ListView.separated(
  itemCount: 100,
  separatorBuilder: (context, index) => Divider(),
  itemBuilder: (context, index) {
    return ListTile(
      title: Text('项目 $index'),
    );
  },
)

常用属性

属性 类型 说明
scrollDirection Axis 滚动方向
padding EdgeInsetsGeometry 内边距
reverse bool 是否反向滚动
shrinkWrap bool 是否根据内容确定高度
physics ScrollPhysics 滚动行为

BottomNavigationBar

BottomNavigationBar 是一个显示在屏幕底部的导航栏，用于在不同的主要视图之间切换。

基本用法

class _MyHomePageState extends State {
  int _selectedIndex = 0;

  static const List _widgetOptions = [
    Text('首页'),
    Text('搜索'),
    Text('设置'),
  ];

  void _onItemTapped(int index) {
    setState(() {
      _selectedIndex = index;
    });
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: const Text('底部导航栏示例'),
      ),
      body: Center(
        child: _widgetOptions.elementAt(_selectedIndex),
      ),
      bottomNavigationBar: BottomNavigationBar(
        items: const [
          BottomNavigationBarItem(
            icon: Icon(Icons.home),
            label: '首页',
          ),
          BottomNavigationBarItem(
            icon: Icon(Icons.search),
            label: '搜索',
          ),
          BottomNavigationBarItem(
            icon: Icon(Icons.settings),
            label: '设置',
          ),
        ],
        currentIndex: _selectedIndex,
        selectedItemColor: Colors.blue,
        onTap: _onItemTapped,
      ),
    );
  }
}

常用属性

属性 类型 说明
items List<BottomNavigationBarItem> 导航项列表
currentIndex int 当前选中项的索引
onTap ValueChanged<int> 点击事件回调
selectedItemColor Color 选中项的颜色
unselectedItemColor Color 未选中项的颜色
type BottomNavigationBarType 导航栏类型

Button

Flutter 提供了多种按钮组件，用于响应用户的点击操作。

基本用法

// ElevatedButton - 带有背景色和阴影的按钮
ElevatedButton(
  onPressed: () {
    // 处理点击事件
  },
  child: Text('Elevated Button'),
  style: ElevatedButton.styleFrom(
    backgroundColor: Colors.blue,
    padding: EdgeInsets.symmetric(horizontal: 24, vertical: 12),
    textStyle: TextStyle(fontSize: 16),
  ),
)

// TextButton - 文本按钮，没有背景色
TextButton(
  onPressed: () {
    // 处理点击事件
  },
  child: Text('Text Button'),
  style: TextButton.styleFrom(
    foregroundColor: Colors.blue,
  ),
)

// OutlinedButton - 带边框的按钮
OutlinedButton(
  onPressed: () {
    // 处理点击事件
  },
  child: Text('Outlined Button'),
  style: OutlinedButton.styleFrom(
    foregroundColor: Colors.blue,
    side: BorderSide(color: Colors.blue),
  ),
)

// IconButton - 图标按钮
IconButton(
  icon: Icon(Icons.favorite),
  onPressed: () {
    // 处理点击事件
  },
  tooltip: '收藏',
)

// 禁用状态
ElevatedButton(
  onPressed: null, // 设置为null表示禁用
  child: Text('禁用按钮'),
)

按钮样式

样式属性 说明
backgroundColor 按钮的背景色
foregroundColor 按钮的前景色（文本和图标）
padding 按钮的内边距
textStyle 文本样式
shape 按钮的形状
elevation 阴影高度（仅 ElevatedButton）
